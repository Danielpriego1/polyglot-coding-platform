# Referencia de Lenguajes Soportados

## Python
- **Comando**: `hero.move_right()`
- **Sintaxis**: Estándar de Python 3.
- **Uso**: Ideal para lógica rápida y scripts simples.

## JavaScript
- **Comando**: `hero.move_right();`
- **Sintaxis**: ES6+ (Node.js).
- **Uso**: Práctica de camelCase y sintaxis web.

## Rust
- **Comando**: `move_right!();`
- **Sintaxis**: Macros de Rust.
- **Uso**: Introducción a la seguridad de memoria y sintaxis de sistemas.

## Go
- **Comando**: `hero.MoveRight()`
- **Sintaxis**: Go 1.18+.
- **Uso**: Práctica de tipado fuerte y convenciones de Google.

## C++
- **Comando**: `hero.move_right();`
- **Sintaxis**: C++17.
- **Uso**: Programación de bajo nivel y alto rendimiento.
